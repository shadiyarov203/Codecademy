HTML Table Reference

Important Table tags

Tag Name Description
<table> Table Creates table
<thead> Table Head The wrapper element for headers in table.
<tr> Table Row Creates a new row in table
<th> Table Header Creates a header in table
<td> Table Data Creates a cell for table data.
<tbody> Table Body The wrapper element for the body of a table.
<tfooter> Table Footer The wrapper element for the footer of a table.
